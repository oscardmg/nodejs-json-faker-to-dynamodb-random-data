Se utilizo el paquete JSON Schema Faker

# Como se utiliza

1. Clonar el repo
2. Ubicarse en la carpeta raiz y ejecutar:

```
yarn
```

3. Crear el archivo .env y asignar los valores de aws para poder conectar

```
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_SESSION_TOKEN=
```

4. Ejecutar:

```
yarn start
```
